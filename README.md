# BBBOXSelector
## Tool 概要
Maya用スクリプトTool。  
バウンディングボックスを表示し、対象物の頂点やフェースを選択するスクリプトToolです。

## インストール・起動方法
- BBBOXSelectorフォルダをMayaのpythonパスが通っているフォルダに格納
- maya上で以下起動スクリプトを実行
  ```python
  import BBBOXSelector
  BBBOXSelector.show()
  ```
## 使用方法
### GUI解説
- 「選択用キューブを作成」ボタン
  - 範囲選択用バウンディングボックスとして使用するポリゴンキューブを生成します。
- 「選択用キューブを削除」ボタン
  - 上記で生成したポリゴンキューブを削除します。
- 各種設定項目
  - Selector Cube
    - バウンディングボックスとして指定するポリゴンキューブノードを指定します。  
    - Set Selectedボタンで、選択中のノードを指定することができます。
    - 基本的に、上記の「選択用キューブ」を指定することを推奨します。
  - Target Mesh
    - 選択対象のメッシュを指定します。
    - Set Selectedボタンで、選択中のノードを指定することができます。
  - Select Type
    - 頂点選択・フェース選択かモードを切り替えます。
  - Tolerance
    - 選択閾値
  - Fully Inside Faces Mode
    - 選択制度の向上
    - バウンディングボックス内側を正確に選択するようにします。
    - ※処理時間が増大します。
- 「内側を選択」ボタン
  - 設定内容に従って、選択処理を実行します。

# 使用する際の注意点
- バウンディングボックスのポリゴンキューブは頂点移動の編集はせず、スケール・移動・回転でのみ修正してください。
  - cubeの現在のスケールや移動・回転から四隅を推定する処理です。
  - 頂点編集やフリーズなどしてcubeのベースサイズや形が変わってしまうと、狙った挙動が取れなくなります。
